UserVSComputerWindow

User VS User game window. Shows: game board, buttons with available moves when its Users turn, winner information when a player wins, "Back to menu" button. User can play moves clicking corresponding buttons. After every move game board is immediately refreshed, then immediately Computer plays his move and board is refreshed again.

ComputerVSComputerWindow

Computer VS Computer game window. Also called Simulation. Shows: game board, Pause/Resume button, winner information when a player wins, "Back to menu" button. Computer players play moves alternately. There is added a fixed delay speed between the moves. Immediately after every move the board is refreshed. Simulation can be paused and resumed any time.
